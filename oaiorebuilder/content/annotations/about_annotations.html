LORE: Literature Object Re-use and Exchange

LORE aims to support the digital practice of researchers and teachers of literary cultures through Annotations, enabling them to:

  • Contribute comments, notes and additional information by annotating web resources.
  • Search and browse annotations contributed by others.

Annotations

LORE aims to enable communities of experts to collaboratively select, tag and annotate digital resources with:

  • keywords
  • notes
  • comments
  • interpretations
  • queries
  • links to related resources etc.
and to share these annotations with the research community to enrich the collection and enhance discovery services.

Getting started with LORE Annotations

You can configure LORE's features via the preferences. Select the icon from the toolbar to show the preferences window. You can enter your name (used to identify you as the creator of annotations), and enable or disable the LORE's features via the General preferences.

Expert users have the option to configure the details of the Annotation service.

For more information about LORE, please see the Aus-e-Lit project page

Logging In and Out

If using the default Annotation server, you can login and logout from the bottom of the Annotation Server Site.
