LORE: Literature Object Re-use and Exchange

LORE aims to support the digital practice of Literature scholars and teachers through:

  • Compound-object authoring:
    • Gather web resources and organise and tag them, describe links between them and make notes about them using a graphical editor.
    • Discover compound objects that others have created about related topics and web resources.
    • Publish the information that you have gathered as compound objects so that others can find them and reuse the information.
    • Present the information that you have gathered and discovered using automatically generated slideshows and diagrams.
  • Annotations:
    • Contribute comments, notes and additional information by annotating web resources.
    • Search and browse annotations contributed by others.
  • Text-mining: Highlight names of people, places and events within the text of the current web resource.

Text-mining

LORE uses the Calais semantic web service to identify names of people, places and events on the current page. Select the Calais button in the toolbar and LORE will highlight the terms identified by Calais in the main browser window.

Right-clicking on the highlighted terms reveals a menu that provides options for searching using these terms, or to use them within LORE to tag compound objects or annotations.

For more information about LORE, please see the Aus-e-Lit project page
